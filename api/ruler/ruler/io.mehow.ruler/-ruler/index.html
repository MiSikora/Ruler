Ruler

object Ruler : LengthConverter, LengthFormatter
Content copied to clipboard

A central point for default Length formatting.

Formatting behaviour can be controlled by installing custom converter factories or formatter factories. If no factories are installed Ruler uses built-in factories to convert and format input.
