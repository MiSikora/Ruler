Meterâ€“based numerical representation of how far apart two points are, such as '150.5 meters'. This class can hold negative values to represent the direction.

This class models a quantity of space in terms of meters and nanometers. Physical distance could be of infinite length. For practicality, the can have values between Long.MIN_VALUE meters exclusive and Long.MAX_VALUE meters and 999_999_999 nanometers inclusive, which is approximately -975 and 975 light years. An attempt to create a distance out of these bounds will throw an exception.
